## Deskripsi Tugas
Repositori ini berisi implementasi 5 algoritma pengurutan data (sorting), yaitu:
1. Bubble Sort
2. Selection Sort
3. Insertion Sort
4. Quick Sort
5. Merge Sort

Kode program ditulis menggunakan dua bahasa pemrograman: **Python** dan **C++**. 
Program dirancang untuk meminta input berupa jumlah data dan nilai data dari pengguna, kemudian menampilkan proses pengurutan secara bertahap (per iterasi) hingga menghasilkan data akhir yang sudah terurut. Repositori ini juga dilengkapi dengan *screenshot* hasil eksekusi (demo) dari kedua program tersebut.
